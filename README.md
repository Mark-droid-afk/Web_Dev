# Mark Jade – Portfolio

A clean, fast, and modern personal portfolio showcasing About, Skills, Projects, Video, Resume link, and Contact. Built with HTML, CSS, and JavaScript. Fully responsive with sticky navigation and a parallax section.

## Features
- Sticky global navigation (desktop + mobile)
- Parallax scrolling section
- Embedded video (YouTube iframe)
- Responsive layout across mobile, tablet, and desktop
- Clear sections: About, Skills, Projects, Resume link, Demo in Youtube,Contact

## Getting Started
### Run locally
1. Clone the repo
```bash
git clone https://github.com/your-username/Web_Dev.git
cd Web_Dev
```
2. Open `welcome.html` in your browser (double-click or use a local server).
   - Optional (recommended): start a local server for better routing and caching or downlaod the live server in VSC extensions.
```bash
# Python 3
python -m http.server 5500
# Then visit http://localhost:5500 to go to the main site   
```

## Project Structure
- `welcome.html` - Welcome Page before main site
- `index.html` – Main site structure
- `style.css` – Styles, sticky header, parallax, responsive video
- `mediaqueries.css` – Additional responsive tweaks
- `script.js` – Menu toggle for mobile, minor interactions
- `assets/` – Images, icons, and PDFs
